Capture Thumbnail


Sample embed to enable captureThumbnail. Note you would want to supply a vs 
vWidget.embed( {
	'targetId': 'vidiun_player',
	'wid': '_243342',
	'uiconf_id' : '12905712',
	'entry_id' : '0_uka1msg4',
	'flashvars': {
		'vs': /* your vs with thumbnail capture permission */
		'captureThumbnail':{
			'plugin': true,
			"tooltip": "Capture Thumbnail"
		}
	}
});
