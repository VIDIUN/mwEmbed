Captions API

The Vidiun captions API enables you to hide and show captions via javascript.
Toggle Captions 
vdp.sendNotification( 'showHideClosedCaptions');

Log caption events:

vdp.vBind('ccDataLoaded', function(){
	$('#log').append( "ccDataLoaded -- All captions has been loaded<br>" );
});
vdp.vBind('newClosedCaptionsData', function(){
	$('#log').append( "newClosedCaptionsData -- A caption asset had been loaded<br>");
});
vdp.vBind('changedClosedCaptions', function(){
	$('#log').append( "ccChanged -- Caption asset as been changed"	);
});
