Vidiun Playlist api


Playlists API:

Switch to Tab index 1 ( electric sheep, 2nd tab listed ) 
// Set the tab index ( select playlist )
vdp.setVDPAttribute('tabBar', 'selectedIndex', 1);
Play second item 
// Play the second clip of the current playlist: 
vdp.setVDPAttribute("playlistAPI.dataProvider", "selectedIndex", 1 );
Get Current Clip Index 
// get current selected clip index:
vdp.evaluate( "{playlistAPI.dataProvider.selectedIndex}" ) );
Get Playlist Data 
// get playlist data provider ( good for building external clip list )
vdp.evaluate("{playlistAPI.dataProvider}") ) );
