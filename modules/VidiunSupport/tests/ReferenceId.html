Vidiun Entry by Reference ID

Vidiun allows to set reference Id per entry. You can use reference Id to set custom id for vidiun entry.
This test allow to load the player with reference Id (without entry Id) and to use changeMedia with reference Id
Note: if you include both entry_id and reference Id the player will use the entry id.

How to use:
Embedding with reference Id: 
vWidget.embed({
	'targetId': 'vidiun_player',
	'wid': '_243342',
	'uiconf_id' : '12905712',
	'flashvars':{
		'referenceId': 'coffee'
	}
});
Dynamic change media via Reference Id on active player: 
document.getElementById( 'vidiun_player' ).sendNotification( 'changeMedia', { referenceId : "vidiun_promo_video" } );
