vWidget Direct JS Server API

vWidget Direct Server API enables simple object request API calls. This should not be confused with the JavaScript client library, which offers object mappings and works with the code generated in the test me console.

The vWidget Direct Server API is included by default every time you include the player library.

Your html5 library include is given to you by the VMC embed dynamic embed code, and looks like this: 
<script src="http//cdnapi.vidiun.com/p/{partnerId}/sp/{partnerId}00/embedIframeJs/uiconf_id/{uiconf_id}/partner_id/{partnerId}"></script>

Get entries from playlist id

Retrieves all the entries for a given playlist id via playlist execute 
<script>
$('#plbutton').click(function(){
	// Create new api object and issue request
	new vWidget.api( {
		'wid' : '_243342',
	}).doRequest({
		'service':'playlist', 
		'action': 'execute', 
		'id': '1_e387kavu'
	}, function( data ){
		// output formatted json result:
		$('#playlist_result').html( 
			syntaxHighlight( data )
		);
	})
});
</script>



Get entry contextData and custom metadata ( multiRequest )

Retrieves all the sources for a given entry id via 
baseEntry getContextData

 and via multi-request list entry custom metaData



Also see vWidget.getSources.html for simple library that builds usable playMainfest source urls 


<script>
$('#entriesbutton').click(function(){
	new vWidget.api( {
		'wid' : '_243342',
	}).doRequest([
		{
			'contextDataParams' : {
				'referrer' : document.URL,
				'objectType' : 'VidiunEntryContextDataParams',
				'flavorTags': 'all'
			},
			'service' : 'baseentry',
			'entryId' : '0_uka1msg4',
			'action' : 'getContextData'
		},
		{
			 'service' : 'metadata_metadata',
			 'action' : 'list',
			 'version' : '-1',
			 // metaDataFilter
			 'filter:metadataObjectTypeEqual' :1, /* VidiunMetadataObjectType::ENTRY */
			 'filter:orderBy' : '+createdAt',
			 'filter:objectIdEqual' : '0_uka1msg4',
			 'pager:pageSize' : 1
		}
	], function(data ){
		$('#entries_result').html( 
			syntaxHighlight( data )
		);
	})
});
</script>
